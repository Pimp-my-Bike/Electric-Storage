## 📋 תיאור הפרויקט / Project Description

מאגר מקיף ומפורט של רכיבים אלקטרוניים עם מידע טכני, מפרטים, חיבורים ודוגמאות קוד. הפרויקט כולל 16 עמודים מפורטים המכסים את כל תחומי האלקטרוניקה.

A comprehensive and detailed repository of electronic components with technical information, specifications, connections, and code examples. The project includes 16 detailed pages covering all areas of electronics.

## 🔧 רכיבים מכוסים / Covered Components

### 1. **רכיבים בסיסיים / Basic Components**
- **נגדים / Resistors** - צבעים, ערכים, סבילות
- **קבלים / Capacitors** - סוגים, ערכים, מתח עבודה
- **טרנזיסטורים / Transistors** - NPN, PNP, MOSFET, IGBT
- **מעגלים משולבים / ICs** - Op-Amps, Logic Gates, Microcontrollers

### 2. **חיבורים ותקשורת / Connections & Communication**
- **מחברים / Connectors** - USB, Audio, DC Power, Headers
- **חיישנים / Sensors** - טמפרטורה, מרחק, תנועה, אור
- **רכיבי RF / RF Components** - WiFi, Bluetooth, LoRa, ZigBee
- **מודולי תקשורת / Communication Modules** - RS485, CAN, Ethernet

### 3. **כוח ושליטה / Power & Control**
- **ספקי כוח / Power Supplies** - Linear, Switched-mode, Battery
- **מנועים / Motors** - DC, Stepper, Servo, Brushless
- **צגים / Displays** - LCD, OLED, TFT, E-Ink

### 4. **כלים וציוד / Tools & Equipment**
- **כלי מדידה / Measurement Tools** - Multimeters, Oscilloscopes
- **כלי עבודה / Hand Tools** - Screwdrivers, Pliers, Cutters
- **כלי חשמל / Power Tools** - Drills, Saws, Soldering Stations

### 5. **תשתיות ואבטחה / Infrastructure & Security**
- **כבלים ומוליכים / Cables & Wires** - חתכים, צבעים, שימושים
- **רכיבי אבטחה / Security Components** - Fuses, Circuit Breakers, Surge Protectors
- **רכיבי זיכרון / Memory Components** - EEPROM, Flash, SD Cards, RAM
